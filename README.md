# cyproject  
Major Project 
Secure Web Application Development is a project focused on building a basic web app with user Sign-Up and Login features while ensuring strong security. The application validates all user inputs, securely stores passwords using hashing, and manages user sessions safely after login. It also protects against common web attacks like SQL Injection, Cross-Site Scripting (XSS), and Cross-Site Request Forgery (CSRF). By following secure coding practices and using technologies like HTML, CSS, JavaScript, backend frameworks, and databases, the project demonstrates how to develop safe and reliable web applications that protect user data and privacy.
